
<h1>create-zely</h1>
a Tool creating zely app

---

> This package is for zely@2 (or lower). To create a zely@3 app, use `zely-cli`.
> ```bash
> npx zely-cli init --dir=app --template=typescript
> ```

 ---


```bash
> npx create-zely

? Project name: my-app
? Directory: backend
? Template: typescript

Cloning Templates...

  $ cd my-app
  $ npm i
  $ npm run dev

Happy Hacking!
```
